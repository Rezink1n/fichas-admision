# Fichas de Admisión

Web app para gestión de fichas de consumición.

## Estructura
- `index.html` — HTML principal
- `css/style.css` — Estilos
- `js/config.js` — Supabase, auth, sesión
- `js/app.js` — Bootstrap, login, setup, navegación
- `js/fichas.js` — Buscar, vender, usar, detalle
- `js/productos.js` — CRUD productos, generar ficha
- `js/establecimientos.js` — Gestión establecimientos
- `js/usuarios.js` — Gestión usuarios/admins
- `js/ui.js` — Modales, QR viewer, edición
- `js/utils.js` — Caducidad, colapsables, utils, vista pública
